Structr 2.1.0 - Enterprise Edition
==================================
Welcome to the Structr Application Platform based on Neo4j. You're running the
Structr Enterprise Edition with the Neo4j Enterprise Bundle included.


Running on UNIX-based systems
-----------------------------
Use bin/start to start Structr.
Use bin/stop to stop it.

Structr will run in the background once it is started.


Running on Windows
------------------
See https://support.structr.com/article/258


Server Logs
-----------
Structr logs to the logs/server.log file.


Learn more
----------
See https://support.structr.com/ for more information.
